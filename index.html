How to publish as GitHub Pages

  1. Create a GitHub repo and push this folder.
  2. In repo Settings → Pages, set Branch to main and Folder to /(root), then save.
  3. Your site will be served at https://USERNAME.github.io/REPO/.

Optional: add a CI workflow to validate mapping.json on every PR.
